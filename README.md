## Use
### Docker
1. [Download config.yml](https://raw.githubusercontent.com/tpc3/Bocchi-Go/main/config.yml)
1. Enter your token to config.yml
1. `docker run --rm -it -v $(PWD):/data ghcr.io/tpc3/bocchi-go`
